# cumt_class_timetable
根据JSON生成课程表ics文件，用于各日历平台导入。
# usage
主函数里的`kcbJson`替换为自己从教务系统抓出的课程表XHR请求返回JSON，运行程序即可生成kcb.ics文件，再手动导入到需要的日历即可。
